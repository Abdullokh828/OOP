Add README.md# OOP — Объектно-ориентированное программирование (C++)

Репозиторий содержит **практические работы и примеры** по курсу **Объектно-ориентированного программирования** на языке C++.

## Структура репозитория

- **Объекты и классы** — базовые понятия ООП, создание классов, конструкторы, деструкторы
- **Наследование** — одиночное и множественное наследование, порядок вызова конструкторов
- **Виртуальные функции** — полиморфизм, абстрактные классы, чисто виртуальные функции
- **Перегрузки** — перегрузка функций и операторов
- **Указатели** — работа с указателями, динамическая память
- **Функции** — различные примеры функций
- **Потоки и файлы** — работа с файловыми потоками (`fstream`, `ifstream`, `ofstream`)
- **STL** — использование стандартной библиотеки шаблонов (контейнеры, алгоритмы и т.д.)
- **ТЗ_Благотворительный фонд.docx** — техническое задание на итоговый проект

## Язык
**C++** (100%)

## Цель репозитория
Сборник лабораторных работ, упражнений и примеров для закрепления знаний по ООП.

---

### Как пользоваться
Каждая папка содержит примеры кода по соответствующей теме. Можешь открывать `.cpp` файлы и запускать их для изучения.
